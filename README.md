# TUYA Ride-Sharing App (Sudan) 🇸🇩

تطبيق حجز رحلات متكامل مصمم خصيصاً للسوق السوداني.

## الميزات:
- تحديد موقع الانطلاق والوجهة عبر الخريطة.
- حساب التسعيرة تلقائياً (300 جنيه أساسي + 500 جنيه لكل كم).
- نظام تواصل فوري (Socket.io).
- واجهة مستخدم باللغة العربية.

## هيكل المشروع:
- `backend/`: خادم Node.js مع Express و Socket.io.
- `frontend/`: واجهة React مبنية بـ Vite.

## طريقة التشغيل محلياً:
1. `cd backend && npm install && node index.js`
2. `cd frontend && npm install && npm run dev`
